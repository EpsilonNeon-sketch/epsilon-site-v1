Quick Start Guide

1. Include the API Library

<script src="https://cdn.jsdelivr.net/npm/@supabase/supabase-js"></script>
<script src="https://epsilon-site-v1.vercel.app/epsilon-api.js"></script>

2. Initialize the API

const api = new EpsilonAPI('YOUR_API_KEY');

3. Make API Calls

// Get user by username
const user = await api.getUserByUsername('warren_epsilon');

// Get user posts
const posts = await api.getUserPosts('warren_epsilon', 10);

// Get all users
const users = await api.getAllUsers(1, 20);

// Get followers
const followers = await api.getUserFollowers('warren_epsilon');

Available Endpoints

  • getUserByUsername(username) - Returns user profile data
  • getUserPosts(username, limit) - Returns user's posts
  • getAllUsers(page, limit) - Returns paginated list of users
  • getUserFollowers(username) - Returns user's followers
  • getUserFollowing(username) - Returns users that user follows
